# Ghana Market Pulse (GMP)

A stock market prediction app for Ghana Stock Exchange.

## Tech Stack

- **Backend**: Flask, Python
- **Frontend**: HTML, CSS, JavaScript
- **Database**: SQLite
- **ML**: scikit-learn, ARIMA

## Installation
```bash
git clone https://github.com/darlingtonerzuah/gmp-market-predictor.git
cd gmp-market-predictor
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## Features

- Live stock dashboard
- ML price predictions
- Price charts
- Community chat

## How to Run
```bash
python dashboard.py
```

Then visit `http://localhost:5000`
